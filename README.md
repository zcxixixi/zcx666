# Abyss Signal

A standalone neon arena browser game.

## Play Local

```bash
python3 -m http.server 4173
```

Open:

```text
http://127.0.0.1:4173
```

## Controls

- `Enter` or click: start
- `T`: toggle autoplay
- `WASD` / arrow keys: move
- Mouse: aim
- `Shift`: dash
- `P`: pause
- `R`: restart
- `F`: fullscreen
